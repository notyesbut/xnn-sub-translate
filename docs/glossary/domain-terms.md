# Domain Terms

## Subtitle Rescue

The product name and overall workflow label.

## Project

One unit of work that contains imported assets, subtitle segments, jobs, flags, glossary terms, and exports.

## Segment

One subtitle event with timing and text fields tracked through import, translation, review, and export.

## QA Flag

A machine-detected warning that suggests a segment may need review.

## Global Offset

A single timing shift applied to every subtitle segment.

## Stretch

A linear timing remap used when subtitle pacing differs across releases.

## Anchor

A user-defined alignment point used to improve sync across the timeline.

## OCR

Optical character recognition used to extract text from image-based subtitle formats.

## Natural Subtitle English

English text optimized for subtitle readability rather than literal word-for-word phrasing.
