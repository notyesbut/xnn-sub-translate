# Decision Log

## 2026-03-25

### D-0001

Use a lightweight monorepo with Tauri + React for the desktop shell, a local Python engine for subtitle processing, and shared contracts for the boundary between them.

### D-0002

Build the text-subtitle MVP first. OCR and advanced sync are planned, but intentionally deferred until the core import, translation, review, and export loop works well.

### D-0003

Keep the product local-first and desktop-first. Do not add mandatory server infrastructure for MVP.
